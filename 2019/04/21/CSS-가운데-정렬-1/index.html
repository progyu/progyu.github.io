CSS-가운데 정렬(1)-margin auto

2019-04-21

  • 주황색 박스를 제외한 오른쪽 공간이 auto 값으로 설정할 수 있는 margin의 최대 영역을 의미한다.
  • 주황색 박스가 block 요소(div)이기 때문에 한 줄 전체 영역을 소유한다. 그러므로 오른쪽 공간은 쓰지 않은 것뿐 주황색 박스가 소유한 공간이다.
  • block 요소는 한 줄에 하나씩 블럭이 쌓이듯 배치된다.

  • text-align: center;

    • block 요소 안에 있는 inline 요소를 가운데 정렬 시킨다.
